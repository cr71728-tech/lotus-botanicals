# botanicals

Static site for Lotus Botanicals LLP.

What this repo contains:
- index.html, about.html, business.html, team.html, contact.html
- License: Apache-2.0 (file: LICENSE)
- .gitignore: Python

Usage
- Work on the default branch: "JUST LANKELAPALEM"
- To preview locally, open index.html in a browser or use a static-server (e.g., `python -m http.server`)

Deployment
- This site can be published with GitHub Pages (serve from the repository root) or any static hosting provider.
